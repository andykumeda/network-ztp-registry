# Security And Sanitization

This repository is a public portfolio version. It intentionally excludes:

- private Git history
- real serial numbers and asset tags
- real building, room, and deployment mappings
- internal IP ranges, hostnames, and domains
- production runbooks and handoff notes
- vendor portal or ITSM instance URLs
- credentials, password hashes, vault files, and SSH keys
- vendored Ansible collections and generated operational readmes

Before publishing changes, run a repository-wide search for prior organization
names, internal network ranges, real serial-number prefixes, former integration
vendor names, personal usernames, and real asset-tag formats. The search should
return no matches outside intentionally synthetic demo data.
